## 5. Merged: connector-clutter branch + macOS CI

`claude/diagram-connector-clutter-uioc6l` was merged into this branch (commit
`2a23f05`). It was authored in a separate cloud session and is likewise **not
compile-verified**. Its changes are independent of the items above (different
code regions — the git merge was clean), but they overlap the SAME subsystems
(connectors, captions, style-panel visibility), so exercise them together.

New/changed tests it brings: `EdgeGeometryTests` (anchor slots / nearest-slot /
route-intersection), `CaptionModeTests` (mode round-trip), and UITest
`step30`–`step32` + an updated `step28`. Verify on a Mac:

- **Relocatable anchors** — drag a connector endpoint around a block; it snaps
  to discrete per-face slots and stays pinned (opts out of auto fan-out); drag
  off-block still detaches.
- **Caption modes** (View ▸ Connector Captions: Always / On Focus / Off) —
  persist in board.json `extra`, honored by PNG/SVG export, undoable; a density
  nudge suggests On Focus past ~40 connectors (shown once per board id).
- **Rubber-band precision** — a band over a connector's bbox gap (not the line)
  no longer selects it.
- **Style-panel visibility** — ⚠️ this branch makes the panel HIDE on deselect,
  which is the OPPOSITE of the sticky-panel behavior on `feature/v0.10-remaining`
  (F6/F7/F8 assume it stays open). Reconcile before combining the two branches
  (see the topology note below).
- **macOS CI** — `.github/workflows/ci-macos.yml` runs the battery; `claude.yml`
  runs `@claude`. Confirm they run on a real macOS runner.

Risk areas flagged by that branch (unverified): tuple-label coercion and
optional-pattern issues in `EdgeGeometry.anchorSlots`/`nearestAnchorSlot` and
`Board+Captions.swift`; `drawEdge(emphasized:)` default hiding captions only in
Off mode for overlay passes; density "shown once" in UserDefaults.

## Branch topology (now unified)

`feature/v0.10-feedback` is the single integration branch. It contains, in order:
the other agent's undo fix, the 22-item batch, the connector-clutter merge +
macOS CI, and the v0.10 follow-ups (link-wipe fix, F6 text size, F7 SVG panel,
B3 panel-pan, F8 z-order, F4 broken links, B10-a routing, B2 caption flicker) —
merged from `feature/v0.10-remaining`.

**Merge reconciliations to eyeball on the Mac** (resolved textually here, but
uncompiled): (1) `drawEdge`/`drawEdgeContent` now carry BOTH `emphasized:`
(clutter caption modes) and `edgeID:` (B2 caption cache); (2) the caption block
gates on `shouldDrawCaption(emphasized:)` and, when it draws, uses the B2
settled-center cache; (3) `refreshStylePanel` keeps the `.image` mode + F8
readout AND adopts the clutter hide-on-deselect (compatible with the B6 sticky
shape tool, which keeps `case .shape` showing the panel); (4) `mouseMoved`
runs both the caption hover-peek and the broken-link hover.
